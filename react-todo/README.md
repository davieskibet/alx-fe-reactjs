# React Todo App

This is a simple React project built with Vite that allows users to:

- Add new todo items
- Toggle todos as completed or not completed
- Delete todos
- Includes comprehensive tests using Jest and React Testing Library

## Features

- React 19 with functional components and hooks
- Vite for fast development and build
- ESLint with recommended rules for React and hooks
- Tailwind CSS for styling
- Jest and React Testing Library for testing components
